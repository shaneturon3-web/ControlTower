# Per-project handoff packets

Each indexed slug has a Drive-synced Architect packet:

```
projects/<SLUG>/
  CLOUD_AI_ENTRY.md      # Start here (Cloud AI)
  EXECUTION_POINTER.md   # ~/Projects path (Cursor Builder)
  project-start.md
  planning/
  references/
```

Created by `shipyard new` or `shipyard register --all`.

**NotebookLM:** Each slug includes `NOTEBOOKLM_ENTRY.md`. Manual: `04_GLOBAL_RULES/NOTEBOOKLM-ARCHITECTURE-SPECIFICATION.md`.
